# 刘培宇 个人主页 — 设计文档

## 概述

刘培宇（Cuyler Liu / Kyler Liu）的个人主页，面向保研申请场景，以垂直滚动分区布局展示成绩、科研论文、竞赛荣誉、英语能力、实习经历及摄影爱好等信息。纯前端单页应用，支持中英文切换。

## 风格

- **配色：** 背景 #F5F6F8（浅灰），白色半透明玻璃拟态卡片（rgba(255,255,255,0.72)），主题色 RGB(203,224,241) 浅蓝及其深色变体 #8BB8D3，文字 #2C2C2C / #6B7280 / #9CA3AF
- **风格关键词：** 玻璃拟态（Glassmorphism）、简约、留白、摄影画廊感
- **装饰元素：** 全屏固定背景网格（60px 间距）+ 三颗径向渐变模糊圆形光斑
- **交互细节：** 卡片悬停上浮 4px + 阴影加深；IntersectionObserver 驱动滚动渐入动画；导航栏向下滚动隐藏 / 向上滚动显示；移动端汉堡菜单

## 技术方案

- 纯前端：HTML + CSS + JS，零依赖/零构建工具
- 单页应用，所有内容在 `personal-homepage/` 目录下
- **i18n 系统：** `i18n.js` 维护中英文词典对象（各 100+ key），通过 `data-i18n` 属性标记 DOM 元素，`setLang()` 遍历替换文本；数字（分数、排名）直接写在 HTML 中不经 i18n
- **动画系统：** CSS `@keyframes fadeUp` + IntersectionObserver 控制卡片滚动渐入
- **导航栏：** `position: fixed` + `backdrop-filter: blur(20px)` 毛玻璃效果，JS 侦听 scroll 实现向下隐藏/向上显示
- **移动端：** 汉堡菜单按钮 toggle `nav--open`，导航链接点击自动关闭
- 头像使用本地图片

## 响应式断点

| 断点 | 布局变化 |
|------|----------|
| > 1024px | 3 列卡片网格，摄影区 1.6:1 双栏 |
| 768–1024px | 2 列卡片网格，摄影区单栏 |
| < 768px | 1 列卡片，汉堡菜单激活，Hero 缩小，时间线竖向排列 |
