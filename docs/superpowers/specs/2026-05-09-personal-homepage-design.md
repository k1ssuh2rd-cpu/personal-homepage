## 概述

刘培宇的个人主页（Personal Home Page），面向保研申请场景，以卡片式布局展示成绩、科研论文、竞赛奖项、英语能力、社会活动等信息。

## 风格

- **配色：** 白色（#FFFFFF）背景 + 灰色（#333 / #666 / #EEE）文字与边框 + RGB(203,224,241) 浅蓝作为主题点缀色
- **风格关键词：** 简约、干净、留白充分
- **交互细节：** 卡片悬停轻微上浮 + 阴影加深，导航栏滚动固定，移动端自适应

## 功能需求

### 1. 多语言切换（中 / 英）

- 页面顶部导航栏右侧放置语言切换按钮
- 所有文本通过 i18n 对象（JS 字典）管理
- 切换即时生效，无页面刷新
- 默认跟随中文

### 2. 卡片交互

- 悬停时轻微上浮 + 阴影过渡动画
- 部分卡片可点击展开查看详情（如论文详情、课程列表）

### 3. 响应式

- 桌面端 3 列卡片
- 平板端 2 列
- 手机端 1 列
